# TC-Death-Chest
Automatically creates a secure chest at player death, safely storing items with location display and elapsed time tracking
This addon automatically creates a secure chest at your death location, safely storing all dropped items for easy recovery.

Even if you fall into the void, your inventory is snapshotted and restored inside a generated chest within valid world limits — ensuring nothing is lost.

Each Death Chest displays:

Player name
Exact coordinates
Dimension
Real-time elapsed timer
Supports Overworld, Nether, and The End.
Smart placement system automatically finds a safe block to spawn the chest, including double chests for large inventories.

⚙ Configuration Menu
Administrators can customize the Death Chest display format using the in-game settings item.

Use the special item named:

 
./settingTC

When used by an operator, it opens a configuration menu where you can:

Edit the chest name format
Use custom tags like:
/Player/ /Time/ /X/ /Y/ /Z/ /Dim/
Add new lines using \n
This allows full control over how the Death Chest text appears in your world.
